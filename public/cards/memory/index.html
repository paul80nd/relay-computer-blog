Memory Card

List of articles related to the 'Memory' card

The Memory card holds 32K of RAM and can hold both a program to execute as well as any associated data. It is one of a few ‘unauthenic’ cards within the computer that is forced to use modern integrated circuits (ICs) to perform its role.
